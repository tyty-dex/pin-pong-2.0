Вот краткое описание игры Ping Pong (Настольный теннис):
## 🏓 Обзор игры
Настольный теннис — олимпийский вид спорта, в котором два игрока или две команды по два человека (парная игра) перебрасывают специальный мяч ракетками через стол с сеткой. Цель — ударить по мячу так, чтобы соперник не смог его вернуть согласно правилам.
## 📋 Основные правила

* Подача: Мяч должен сначала удариться о половину стола подающего, перелететь сетку и коснуться половины соперника.
* Розыгрыш: Игроки должны отбивать мяч после одного касания о свою сторону стола. Бить «с лёта» (пока мяч не коснулся стола) нельзя.
* Начисление очков: Очко присуждается, если противник совершил ошибку (не попал по мячу, попал в сетку или в аут).
* Победа: Сет длится до 11 очков. Для победы в сете нужен отрыв минимум в 2 очка. Матч обычно состоит из нечётного количества партий (от 3 до 7).

## ⚙️ Инвентарь

   1. Стол: Стандартный размер — 2,74 м в длину и 1,525 м в ширину.
   2. Сетка: Высотой 15,25 см, разделяет стол пополам.
   3. Ракетка: Изготавливается из дерева, покрытого резиновыми накладками с обеих сторон.
   4. Мяч: Легкий полый шар из пластика (диаметром 40 мм).

## 🚀 Ключевые навыки

* Реакция: Требует мгновенного принятия решений.
* Вращение: Игроки используют «топ-спины» и «подрезки», чтобы изменить траекторию полета мяча.
* Скорость: Это один из самых быстрых видов спорта в мире по скорости движения снаряда относительно размеров площадки.

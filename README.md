Classic Ping-Pong 🏓
Минималистичная аркада для двоих игроков на одном устройстве. Ремейк легендарной классики, где победа зависит от скорости реакции и точности движений.
Особенности
Локальный мультиплеер (PvP): Соревнуйтесь с другом на одной клавиатуре.
Динамическая сложность: Скорость мяча увеличивается после каждого удара, накаляя темп игры.
Автоматический счет: Система фиксирует голы и отображает текущий результат в реальном времени.
Физика столкновений: Реалистичные отскоки мяча от ракеток и границ экрана.
Управление
Игрок 1 (Слева): клава W / S
Игрок 2 (Справа): стрелки ↑ / ↓
Стек технологий
Язык: Python 3.x
Библиотека: Pygame (графика, обработка событий и физика)
Запуск
Установите зависимости: pip install pygame

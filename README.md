🏓 Пинг-понг (Настольный теннис)
Пинг-понг — это динамичный олимпийский вид спорта, в котором два игрока или две команды по два человека соревнуются, перебрасывая специальный мяч ракетками через стол с сеткой.
📋 Основные правила
Подача: Мяч должен сначала удариться о половину стола подающего, перелететь сетку и коснуться половины соперника.
Розыгрыш: Игроки должны отбивать мяч после одного касания о свою сторону стола.
Начисление очков: Очко присуждается, если противник:
Не смог отбить мяч.
Отбил мяч за пределы стола.
Коснулся стола рукой или сдвинул его.
Победа: Партия обычно идет до 11 очков. Матч состоит из нечетного количества партий (например, до 3 или 4 побед).
🛠 Экипировка
Ракетка: Основание из дерева, покрытое резиновыми накладками с обеих сторон.
Мяч: Легкий пластиковый шар диаметром 40 мм (белого или оранжевого цвета).
Стол: Стандартный размер — 2,74 x 1,525 метра, высота сетки — 15,25 см.
💡 Почему стоит играть?
Реакция: Развивает молниеносную координацию и скорость принятия решений.
Доступность: Подходит для любого возраста и уровня физической подготовки.
Безопасность: Минимальный риск травм по сравнению с контактными видами спорта.
"Настольный теннис — это шахматы в движении со скоростью 100 миль в час."
